# Cyber Risk Prediction Platform
### Tamil Nadu Cybercrime Threat Intelligence System

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the application
python app.py

# 3. Open in browser
http://localhost:5000
```

## Features
- 🛡️ Secure login/register with Admin & User roles
- 🗺️ Interactive Tamil Nadu threat map (Green/Yellow/Red)
- 📊 5 live interactive Chart.js visualizations
- 🤖 AI-powered threat pattern analysis (backend ML)
- 🔔 Real-time alert bell + popup notifications
- 📋 Full CRUD incident management (Admin)
- 🎯 Inline edit/update on Data Entry page
- 🛡️ Cyber Safety awareness + Defense Protocols
- 🌌 3D animated cyber background (particle + grid)
- 🏆 District Threat Ranking

## All 10 Modules
1. Secure Access Gateway (Login/Register)
2. Cyber Incident Data Entry (Add/Edit/Update/Delete)
3. Cyber Threat Intelligence Map (color-coded)
4. Threat Pattern Intelligence (AI analytics)
5. Scam Pattern Visualization (5 charts)
6. Incident Insight Viewer (filterable table)
7. Threat Alert Beacon (popup + bell)
8. Cyber Defense Recommendation (per scam)
9. Cyber Awareness Hub (education)
10. District Threat Ranking

## Technology Stack
- **Frontend:** HTML5, CSS3, Bootstrap icons, Chart.js, Orbitron font
- **Backend:** Python Flask
- **ML:** scikit-learn, pandas, numpy (backend only)
- **Storage:** In-memory (no DB setup needed)
